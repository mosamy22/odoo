How to Use

  1. Go to the Odoo Apps menu and search for "Paymob", Install the Paymob plugin. In the list of providers, activate Paymob.
  2. Enter your Paymob credentials based on the mode you selected.

    To get your credentials, go to your Paymob account dashboard and navigate to Settings.

  3. For Paymob to notify Odoo about the payment status, set the Notification URL in your integration from the account dashboard to: <your-website>/payment/paymob/return
  4. To enable tokenized payments (saving cards for future use):

    Go to your Paymob account and navigate to Checkout Customization Enable the Show Save Card option.

  5. Authorized Sales Setup:

    6. If your integration supports is_auth, you can receive authorized transactions in Odoo.
